Explicações sobre Triângulos

Classificação por Lados

Triângulo Equilátero

Todos os três lados têm o mesmo comprimento. Todos os ângulos internos medem 60°.

Triângulo Isósceles

Dois lados têm o mesmo comprimento e dois ângulos são iguais.

Triângulo Escaleno

Todos os três lados têm comprimentos diferentes. Todos os ângulos internos são diferentes.

Classificação por Ângulos

Triângulo Acutângulo

Todos os três ângulos internos são agudos (menores que 90°).

Triângulo Retângulo

Possui um ângulo reto (exatamente 90°).

Triângulo Obtusângulo

Possui um ângulo obtuso (maior que 90°).

Fórmulas Básicas

Área do Triângulo

A = (b × h) / 2

Onde b é a base e h é a altura.

Perímetro do Triângulo

P = a + b + c

Onde a, b e c são os comprimentos dos lados.

Teorema de Pitágoras

a² = b² + c²

Para triângulos retângulos, onde a é a hipotenusa.

Lei dos Cossenos

a² = b² + c² - 2bc·cos(A)

Para encontrar um lado conhecendo dois lados e o ângulo entre eles.
